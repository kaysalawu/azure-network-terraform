Usage: main.py [command] -g RESOURCE_GROUP -c CIRCUIT_NAME

This script requires options and arguments:
  -g, --resource-group   The name of the resource group.
  -c, --circuit-name     The name of the circuit.

Commands:
  list mcr               List all MCRs under the specified resource group and circuit.

Options:
  -h, --helper           Display this help message and exit.

Examples:
  main.py list mcr -g myResourceGroup -c myCircuitName
